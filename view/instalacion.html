Guia de instalación y configuración básica de Odoo

Nota:Esta guía cubre la instalación de Odoo 16 en Windows 10 usando el instalador oficial.

Instalación paso a paso

1. Descargar el instalador de Odoo

  1. Visitar el sitio oficial de Odoo: https://www.odoo.com
  2. Rellenar el formulario con los diferentes datos personales
  3. Descargar la versión Oodo 16 Community para Windows (saltará un mensaje de instalación)
  4. Finalmente, debemos ejecutar el archivo .exe descargado.

2. Asistente de instalador Odoo

  1. Seleccionar el idioma, en nuestro caso "inglés".
  2. Darle click a "Siguiente" para seguir con la guía de instalacion.
  3. Aceptar los términos de licencia.
  4. Seleccionar los componentes a instalar y darle a "Siguiente":
    • Odoo Server
    • PostgreSQL Database
  5. Configurar credenciales de PostgreSQL:
    • Hostname: localhost
    • Puerto: 5432
    • Usuario: openpg
    • Contraseña: openpgpwd (crear una contraseña más segura)
  6. Especificar el directorio de instalación (C:\...), normalmente está por defecto.
  7. Esperamos que se decargue el Odoo, Service, PostgreSQL, etc.
  8. Por último, le damos a "Siguiente" y "Finalizar".
  9. Directamente iremos a Odoo del navegador web por el check, que nos lo ejecuta directamente.

4. Acceder a Odoo

  1. En caso de que no se nos abra, abriremos un navegador web.
  2. Ingresar a: http://localhost:8069
  3. Crear una nueva base de datos:
    • Nombre de la base de datos (nombre de alumno o empleado)
    • Email
    • Contraseña
    • Número de teléfono
    • Seleccionar idioma y país
    • Seleccionar Demo data (✔)
    4. Nota: ¡¡Es importante apuntarse el Master password!!
  4. Ahora iniciamos sesión en la web de Odoo y entramos a las aplicaciones y usuarios.
¡Instalación completada! Ahora podemos comenzar a utilizar Odoo en nuestro entorno Windows 10.
