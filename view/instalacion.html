Nota: Esta guía cubre la instalación de Odoo 16 en Windows 10 usando el instalador oficial.

Instalación paso a paso

1. Descargar el instalador de Odoo

  1. Visitar el sitio oficial de Odoo: https://www.odoo.com
  2. Rellenar el formulario con los diferentes datos personales
  3. Descargar la versión Oodo 16 Community para Windows y nos saldrá luego un mensaje tambien.
  4. Luego de ello hemos de ejecutar el archivo .exe descargado

2. Ejecutar el asistente de instalación

  1. Aceptar el acuerdo de licencia
  2. Seleccionar los componentes a instalar:
    • Odoo Server
    • PostgreSQL Database
  3. Especificar el directorio de instalación (C:\Odoo por defecto)
  4. Configurar credenciales de PostgreSQL:
    • Usuario: postgres
    • Contraseña: (crear una segura)

3. Configurar el servicio de Windows

  1. Durante la instalación, seleccionar "Install as service"
  2. El servicio se iniciará automáticamente al finalizar la instalación
  3. Para verificar, abrir el Administrador de tareas → Pestaña Servicios

4. Acceder a Odoo

  1. Abrir un navegador web
  2. Ingresar a: http://localhost:8069
  3. Crear una nueva base de datos:
    • Nombre de la base de datos
    • Email
    • Contraseña
    • Seleccionar idioma

Configuración inicial

Configuración de la empresa

  1. Ir a Configuración → Empresas
  2. Completar la información de la empresa:
    • Nombre
    • Dirección
    • Logo
    • Información fiscal
  3. Guardar los cambios

Instalar módulos básicos

  1. Ir a Aplicaciones
  2. Buscar e instalar módulos esenciales:
    • Ventas
    • Compras
    • Inventario
    • Contabilidad
  3. Hacer clic en "Instalar" para cada módulo
¡Instalación completada! Ahora puede comenzar a utilizar Odoo en su entorno Windows 10.
