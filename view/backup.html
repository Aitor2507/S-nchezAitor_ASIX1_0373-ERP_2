Guia de backup para Odoo

Instalación paso a paso


1. Acceder a Odoo

  1. Entrar a un navegador e ingresar a: http://localhost:8069/web/database/manager y le damos a backup


  2. Luego de esto, nos pedirá los datos de "Master Password", el nombre de la base de datos y en que formato quieres el .zip.


  3. Vemos que tenemos listo el zip.


  4. Ahora nos dirigiremos a la máquina en la que queremos que este la base de datos y nos dirigimos al mismo link. Aqui le daremos a "Restore Database", donde restauraremos la base de datos a partir del .zip anterior.


  5. Por otra parte, debemos rellenar los datos para la base de datos, en este caso "Master Password", el archivo ZIP que queremos restaurar, el nombre de la base de datos y finalmente seleccionar que se ha removido ya que viene de otro lugar.


  6. Una vez hecho todo lo anterior, ya estaría lista la restauración de la base de datos de "db_sanchezaitor".
